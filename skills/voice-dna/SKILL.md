# Voice DNA - 글쓰기 스타일 추출

대상: $ARGUMENTS

## 분석 프로세스

### 1. 문체 분석 요소

#### 문장 구조
- 평균 문장 길이 (짧은/중간/긴)
- 단문 vs 복문 비율
- 문장 시작 패턴 (주어, 접속사, 부사 등)
- 수동태 vs 능동태 비율

#### 어휘 특성
- 격식 수준 (구어체/문어체/혼합)
- 전문 용어 사용 빈도
- 비유/은유 패턴
- 자주 쓰는 표현/구문

#### 리듬과 흐름
- 문단 길이 패턴
- 전환어 사용 방식
- 강조 기법 (반복, 대조, 열거)
- 호흡 (쉼표, 대시 사용)

#### 감정과 톤
- 감정 온도 (냉정/중립/열정적)
- 유머 사용 여부/방식
- 독자와의 거리감 (격식/친근)
- 확신 수준 (단정적/조심스러운)

### 2. 스타일 프로필 생성

```
## Voice DNA 프로필

**한줄 요약**: [이 글쓴이의 스타일을 한 문장으로]

### 톤
- 격식: [1-5]
- 친근함: [1-5]
- 에너지: [1-5]
- 유머: [1-5]

### 문장 DNA
- 평균 길이: [짧음/중간/김]
- 선호 구조: [단문 위주 / 복문 위주 / 혼합]
- 특징적 패턴: [...]

### 어휘 DNA
- 수준: [쉬운/보통/전문적]
- 자주 쓰는 표현: [...]
- 피하는 표현: [...]

### DO (이렇게 쓰세요)
1. ...
2. ...

### DON'T (이렇게 쓰지 마세요)
1. ...
2. ...

### 예시 변환
- Before (일반적): "..."
- After (이 스타일): "..."
```

### 3. 스타일 적용
- 추출된 Voice DNA를 기반으로 새 글 작성
- 핵심 메시지는 유지하되 문체만 변환
- 원본과 비교하여 스타일 일치도 확인

## 규칙
- 최소 3개 이상의 샘플 텍스트로 분석하면 정확도 향상
- 스타일 모방이지 표절이 아님 - 내용은 새롭게
- 문화적/언어적 맥락 고려
- 프로필은 재사용 가능하도록 파일로 저장
